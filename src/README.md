# 🚀 향상된 다음날 오전 상승 예측 주식 추천 시스템 (글로벌 데이터 + AI 트렌드 분석)

**2025년 4월 AI 전력 인프라 성장**과 **일론 머스크 효과**처럼 새로운 기술/영역이 이슈화되는 순간을 포착하고, **영향력 있는 기관/인물(11대)**의 움직임을 실시간 분석하는 차세대 주식 예측 시스템입니다.

## 🎯 핵심 혁신 기능

### ✅ **새로운 기술/영역 이슈 감지**
- **AI 전력 인프라**: 데이터센터 전력 수요, 가스터빈, 액침냉각, 원자력
- **차세대 AI**: 초거대AI, AGI, 휴머노이드, AI 에이전트
- **에너지 혁신**: 원자력, SMR, 수소에너지, ESS
- **우주 경제**: 우주산업, 위성인터넷, 스페이스X
- **차세대 반도체**: 양자컴퓨터, 광자반도체, HBM4
- **생명공학**: 유전자편집, CRISPR, mRNA
- **블록체인/Web3**: Web3, 탈중심화, NFT, 메타버스

### ✅ **영향력 10대 기관/11대 인물 실시간 분석**
- **기관**: Fed, SEC, OPEC+, 한국은행, 블랙록, ECB, BOJ, MSCI, IMF, 금융위원회
- **인물**: 제롬 파울, 재닛 옐런, 젠슨 황, 일론 머스크, 워런 버핏, 크리스틴 라가르드, 우에다 가즈오, 이창용, 제이미 다이먼, 팀 쿡, **이재명 대통령**
- **영향력 등급**: CRITICAL/HIGH/MEDIUM 단계별 시장 영향 예측

### ✅ **일론 머스크 효과 포착**
- **Signal 사례**: 머스크 언급으로 5600% 급등
- **관련 기업**: 테슬라, 스페이스X, 도지코인
- **예측 변동성**: ±5~15%

### ✅ **국가별 주식 분류**
- **🇰🇷 한국 주식**: 삼성전자, SK하이닉스, 네이버, KB금융 등
- **🇺🇸 미국 주식**: TSMC, NVIDIA, Apple, Microsoft 등  
- **🌍 기타**: 분류되지 않은 기관/용어

### ✅ **하락 리스크 예측**
- **부정적 뉴스 감지**: margin pressure, warning, shrink, concern 등
- **섹터별 위험도**: 금융(15점), 2차전지(12점), 바이오(10점)
- **매도 고려 추천**: 상위 5개 리스크 주식 제안

## 🚀 설치 및 실행

### 1. 의존성 설치
```bash
pip install -r requirements.txt
pip install --upgrade yfinance  # 최신 버전으로 업그레이드 필요
```

### 2. 향상된 시스템 실행
```bash
# 단일 실행 (추천)
python3 enhanced_main.py

# 기존 시스템 실행
python3 stock_ranker_main.py --mode single --output print
```

### 3. 스케줄링 실행
```bash
python3 stock_ranker_main.py --mode schedule
```

### 3. 시장 영향 예측 레벨
- **⚡ CRITICAL**: Fed, 대통령 등 시장 전반 영향 (±2~5%)
- **🔥 HIGH**: CEO, 대형 기관 등 특정 섹터 영향 (±5~15%)
- **💡 MEDIUM**: 부분적 시장 영향 (±1~3%)

### 4. 점수 계산 공식
```
기본 점수 = 언급 횟수 × 10점
감성 보너스 = 긍정 뉴스 +5점, 부정 뉴스 -5점
섹터 가중치 = 동적 가중치 적용 (1.0~1.4배)
최종 점수 = (기본 점수 + 감성 보너스) × 섹터 가중치
```

## 🎯 투자 전략

### 조기 진입 전략 (새로운 트렌드)
1. **AI 전력 인프라**: 가스터빈, 액침냉각, 원자력 관련주 조기 매수
2. **일론 머스크 효과**: 언급 즉시 관련주 매수/매도 결정
3. **영향력 인물 발언**: CRITICAL 등급 시 포지션 조정

### 기존 전략 (고도화)
1. **진입**: 오전 9:00-9:30, 시초가 상승 종목 중심
2. **수익실현**: +5~10%, 오전 11:30까지
3. **손절**: -3~5%, 글로벌 변수 시 강화

### 리스크 관리 강화
1. **분산투자**: 섹터별/국가별 분산
2. **변동성 주시**: 영향력 예측 레벨 모니터링
3. **하락 리스크**: 시스템 제안 매도 주식 고려
